# 功能介绍

## 核心功能

### 日历视图

DailyStuff 以日历为核心入口，支持月视图和周视图切换。

- **月视图**：俯瞰整月，快速定位任意一天
- **周视图**：聚焦当周，适合详细规划
- **日期预览**：点击日期查看当天摘要，无需跳转

### 多媒体日记

每一天都是一个容器，支持多种内容类型：

| 类型 | 描述 |
|-----|------|
| 文字 | 支持 Markdown 格式 |
| 照片 | 支持拍照或从相册导入 |
| 视频 | 支持录制或导入 |
| 音频 | 支持录音 |
| 画廊 | 将多张图片组织为相册 |
| 待办 | 带提醒的任务管理 |
| 位置 | 记录地点信息 |

### 心情追踪

- 5 级心情评分（1-5）
- 自定义心情标签
- 心情趋势可视化

### 待办事项

- 创建待办项目
- 设置提醒时间
- 邮件提醒（可选）
- 周期任务支持
- 逾期任务管理

## AI 功能

所有 AI 功能在设备本地运行，保护你的隐私。

### 图片 OCR
- 自动识别图片中的文字
- 支持中文和英文
- 可通过文字内容搜索图片

### 图片智能分类
- 自动为图片添加标签（如：风景、食物、人物）
- 可通过标签搜索图片

### 语音转写
- 将录音转换为文字
- 基于 whisper.cpp，完全离线
- 支持多种语言

## 数据管理

### 批量导入
- 从相册批量选择媒体
- 自动按拍摄日期分类
- 支持手动指定日期

### 备份与恢复
- 本地备份（保存到指定目录）
- 邮件备份（发送到邮箱）
- 选择性备份（按日期范围）

### 回收站
- 删除的内容暂存回收站
- 可恢复或永久删除

## 搜索功能

强大的全文搜索：
- 搜索文字内容
- 搜索 OCR 识别的图片文字
- 搜索图片标签
- 搜索语音转写内容
- 搜索画廊名称

## 隐私设计

- **离线优先**：核心功能无需联网
- **本地存储**：数据存储在设备本地
- **端侧 AI**：AI 功能在设备上运行
- **无追踪**：不收集用户行为数据
- **可导出**：随时导出所有数据
