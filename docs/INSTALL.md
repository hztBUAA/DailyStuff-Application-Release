# 安装指南

## 下载 APK

从 [apk](../apk/) 目录下载最新版本的 APK 文件。

## 安装步骤

### 方法一：直接安装

1. 将 APK 文件传输到你的 Android 手机
2. 在手机上打开文件管理器，找到 APK 文件
3. 点击安装
4. 如果提示「为了安全，你的手机不允许安装来自此来源的应用」，点击「设置」
5. 允许该来源安装应用
6. 返回并完成安装

### 方法二：通过 ADB 安装（开发者）

```bash
# 连接手机（开启 USB 调试）
adb devices

# 安装 APK
adb install dailystuff-1.0.0-beta.1.apk
```

## 权限说明

应用会请求以下权限：

| 权限 | 用途 | 必需 |
|-----|------|-----|
| 相机 | 拍照、录像 | 可选 |
| 麦克风 | 录音 | 可选 |
| 存储/媒体 | 保存和读取照片、视频、音频 | 是 |
| 位置 | 记录地点 | 可选 |
| 通知 | 待办提醒 | 可选 |

你可以根据需要授予或拒绝可选权限，不会影响核心功能使用。

## 常见问题

### Q: 安装失败，提示「解析包时出错」
A: 请检查：
1. APK 文件是否完整下载
2. 手机存储空间是否充足
3. Android 版本是否满足要求（需要 8.0 以上）

### Q: 安装后打开闪退
A: 请尝试：
1. 重新下载并安装
2. 清除应用数据后重试
3. 在内测群反馈，附上手机型号和 Android 版本

### Q: 如何卸载？
A: 长按应用图标 → 卸载，或在设置 → 应用 → DailyStuff → 卸载

## 数据迁移

如果从旧版本升级，数据会自动保留。如需手动备份：

1. 打开应用 → 设置 → 数据管理
2. 选择「备份」
3. 保存备份文件
