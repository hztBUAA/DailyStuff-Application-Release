# 更新日志

## v1.0.0-beta.1 (2025-01-19)

首个内测版本发布！

### 新功能
- 日历视图（月/周切换）
- 多媒体日记支持
  - 文字笔记（支持 Markdown）
  - 照片（拍照/导入）
  - 视频（录制/导入）
  - 音频（录音）
- 心情追踪（1-5 级 + 自定义标签）
- 待办事项管理
  - 提醒功能
  - 邮件提醒
  - 周期任务
- 画廊功能
  - 创建画廊相册
  - 画廊命名
- 端侧 AI 功能
  - 图片 OCR 文字识别
  - 图片智能分类
  - 语音转写（whisper.cpp）
- 位置记录
- 批量导入媒体
- 数据管理
  - 本地备份
  - 邮件备份
  - 回收站
- 搜索功能
  - 全文搜索
  - AI 搜索（OCR/标签/转写）
