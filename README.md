# DailyStuff - 内测发布

欢迎来到 DailyStuff 内测版！这是一款以日历为核心的私人生活管理 Android 应用。

## 应用介绍

DailyStuff 是一款注重隐私的日记与生活管理应用，核心特点：

- **日历优先**：以日历为主入口，每一天都是你的生活容器
- **隐私至上**：离线优先设计，AI 功能可选且默认关闭，数据不上传
- **轻量存储**：媒体文件存储在系统相册，应用只保存指针引用

## 主要功能

### 已完成功能
- 日历视图（月/周切换）
- 多媒体日记（文字/照片/视频/音频）
- 心情追踪（1-5 级 + 自定义标签）
- 待办事项管理
- 画廊功能
- 批量导入媒体
- 数据备份与导出
- 端侧 AI 功能
  - 图片 OCR 文字识别
  - 图片智能分类
  - 语音转文字（whisper.cpp）
- 位置记录

### 开发中功能
- Widget 小部件
- 深色主题
- 流式录音转写

## 下载安装

### 当前版本
- **版本号**: 1.0.0-beta.1
- **下载地址**: [apk/dailystuff-1.0.0-beta.1.apk](apk/dailystuff-1.0.0-beta.1.apk)

### 安装说明
1. 下载 APK 文件
2. 在手机上打开文件，允许「安装未知应用」
3. 完成安装并打开应用

### 系统要求
- Android 8.0 (API 26) 及以上
- 推荐 Android 11+ 以获得最佳体验
